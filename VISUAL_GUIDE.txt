│  STEP 1: FIX CLERK KEYS (⚠️ CRITICAL - DO THIS FIRST!)          │
└──────────────────────────────────────────────────────────────────┘

   1. Open: https://dashboard.clerk.com
   2. Sign in to your account
   3. Click "API Keys" in sidebar
   4. Copy BOTH keys:
      • Publishable Key (pk_test_...)
      • Secret Key (sk_test_...)
   
   5. Update server\.env:
      ┌────────────────────────────────────────────────────┐
      │ CLERK_PUBLISHABLE_KEY=pk_test_YOUR_NEW_KEY        │
      │ CLERK_SECRET_KEY=sk_test_YOUR_NEW_SECRET_KEY      │
      └────────────────────────────────────────────────────┘
   
   6. Update client\.env:
      ┌────────────────────────────────────────────────────┐
      │ VITE_CLERK_PUBLISHABLE_KEY=pk_test_YOUR_NEW_KEY   │
      └────────────────────────────────────────────────────┘

Problem: "Clerk: Handshake token verification failed"
Solution: Get NEW keys from dashboard.clerk.com
          Keys must be 50+ characters long!

Problem: "Database Connection Error"
Solution: Check MongoDB URI in server\.env
          Ensure MongoDB is running

Problem: "Cannot find module"
Solution: Run npm install in both folders

Problem: "Port already in use"
Solution: npx kill-port 3000
          npx kill-port 5173
